# Team Workflow

## Before Starting Work

Always pull the latest version:

```bash
git pull origin main
```

## Create Your Own Branch

Replace "your-name" with your actual name:

```bash
git checkout -b your-name-feature
```

Examples:

```bash
git checkout -b qiyaam-integration
git checkout -b john-ui
git checkout -b sarah-parser
```

## Save Your Work

```bash
git add .
git commit -m "Completed assigned feature"
```

## Push Your Branch

```bash
git push origin your-name-feature
```

## Create Pull Request

Create a Pull Request into:

main

Do NOT push directly to main.
