# Mini Spreadsheet Frontend Challenge

## Overview

This project is a browser-based spreadsheet application built using:

- HTML
- CSS
- Vanilla JavaScript

The application allows users to enter numbers, text, and formulas into cells while automatically recalculating dependent cells.
